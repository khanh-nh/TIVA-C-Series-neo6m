# TIVA-C-Series-neo6m
This project implements the GPS module Ublox Neo 6m using the Tiva C series TM4C123G launchpad.

 ## Pin map: 

Buzzer (+) -> PF3\
Buzzer (-) -> GND\
LCD (+) -> VBUS\
LCD (-) -> GND\
LCD (SDA) -> PB1\
LCD (SCL) -> PB0\
GPS (TX) -> PB0 (RX)\
GPS (+) -> 3.3V\
GPS (-) -> GND
